  • Allcomp cutter HMI

    "WPF Work Project"

    At Allcomp Polska, I specialize in developing advanced HMI applications for industrial cutting machines. Using C# and .NET technologies, I design and enhance user interfaces with WPF and manage databases through Microsoft SQL Server.

    Key Responsibilities:

    • HMI Development: Crafting user-friendly interfaces for machine control.
    • Trajectory Generation: Creating algorithms for precise cutting paths.
    • Peripheral Management: For example, integrating printer control, managing camera integration and file management.
    • Optimization Algorithms: Developing cutting algorithms and mathematical models.
    • Multi-Display Support: Adding features for additional display elements.
    • PLC Integration: Ensuring smooth communication with PLCs.

    This role combines technical expertise and innovation, contributing to cutting-edge industrial solutions.

  • Multiagent Queue Simulation

    Unity Project

    This project showcases my exploration of multi-agent simulation and the Unity engine to model realistic ticket queue scenarios. It highlights my passion for problem-solving and innovative thinking.

    The simulation examines how people behave in ticket queues and how the number and type of queues affect throughput. It includes both traditional and self-service ticket queues to optimize efficiency and comfort.

    Created using the Unity engine and coded in C#.

    Key Features

    • Simulates traditional and self-service ticket queues.
    • Adjustable parameters: number of queues, guests, and service times.
    • Self-service queues are 30% faster.
    • Up to 40% of guests use self-service if the queue is shorter.

    The project can be expanded to include factors like different age groups, service costs, and customer satisfaction.

  • Real Estate Web Application

    Angular Project

    I developed a web application enabling users to post and browse real estate ads, facilitating communication between buyers and sellers. This project showcases my skills in dynamic web development and security.

    Key features include:

    • Data binding using Angular for dynamic interfaces.
    • Image handling with Cloudinary for optimized delivery.
    • Deployment readiness on Azure with secure settings via Azure Key Vault.
    • Authorization and authentication using IdentityServer4.
    • Exception handling and middleware in ASP.NET Core.
    • Automapper for data mapping and updating.
    • UnitOfWork and Repository patterns for efficient data access.
    • API and EntityFramework for robust data storage.
    • Sorting and filtering functionality for ads.
    • Resolver and property-detail page for detailed ad displays.
    • Reactive forms and validation for user input.
    • HTML form with Angular additions for user registration.
  • ReSCat: Simple ERP Application

    ReSCat WPF Project

    ReSCat is my first project, marking the beginning of my programming journey. It is a basic ERP system developed using WPF, designed to help companies manage and report their manufacturing processes. This project highlights my learning experience in WPF, C#, XAML, SQL, and Entity Framework.

    Key features include:

    • Creation and management of a database for company records.
    • CRUD operations (Create, Read, Update, Delete) for database entries.
    • Data display using DataGrid with search functionality.
    • XML file handling for exporting and importing data.
    • Integrated calendar for production scheduling.
    • Built-in calculator and notepad for quick calculations and notes.
    • Navigation menu for easy access to different sections.
    • Charting and logging capabilities for data visualization and tracking.
    • Exception handling and middleware for robust operations.
    • Responsive UI with custom themes.
    • Data binding and validation using WPF and XAML.
