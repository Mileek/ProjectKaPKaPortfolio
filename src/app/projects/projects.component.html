  • Multiagent Queue Simulation

    Unity Project

    This project showcases my exploration of multi-agent simulation and the Unity engine to model realistic ticket queue scenarios. It highlights my passion for problem-solving and innovative thinking.

    The simulation examines how people behave in ticket queues and how the number and type of queues affect throughput. It includes both traditional and self-service ticket queues to optimize efficiency and comfort.

    Created using the Unity engine and coded in C#.

    Key Features

    • Simulates traditional and self-service ticket queues.
    • Adjustable parameters: number of queues, guests, and service times.
    • Self-service queues are 30% faster.
    • Up to 40% of guests use self-service if the queue is shorter.

    The project can be expanded to include factors like different age groups, service costs, and customer satisfaction.

  • FTP Server

    FTP Server using .NET Sockets

    SimpleServerFTP is a lightweight, C#-based FTP server application designed to support basic file transfer operations. Built on .NET Sockets, it efficiently handles user authentication, file uploads, downloads, directory listings, and other standard FTP commands.

    Key features include:

    • Support for both anonymous and authenticated users.
    • Handling of basic FTP commands like USER, PASS, PWD, CWD, MKD, RMD, PASV, LIST, STOR, RETR, and DELE.
    • Passive mode for data transfers.
    • Configuration via a JSON file.

    The server is configurable through a JSON file, allowing customization of IP address, port number, user authentication, and directory settings.

    SimpleServerFTP demonstrates my proficiency in C# and .NET Core, and my ability to create functional and configurable server applications.

  • FTP Client

    FTP Client using .NET Sockets

    SimpleClientFTP is a lightweight FTP client implemented in C#. It facilitates basic FTP operations such as directory listing, file uploads, downloads, directory creation, deletion, and passive mode support. The client leverages .NET sockets for efficient network communication, ensuring reliable data transfers.

    Key features include:

    • Supports connection to FTP servers with authentication.
    • Allows listing directories and files.
    • Enables file uploading and downloading.
    • Facilitates directory creation and deletion.
    • Includes passive mode support for data transfers.
    • Can be used as a batch or console application for automation tasks.

    SimpleClientFTP utilizes the ClientFTP class to manage connection setup, command execution, and data transfer operations. It is designed for use in batch files and scripting scenarios, enhancing its versatility for automated tasks.
